# Cinematic React Portfolio

A multi-page portfolio built with React, TypeScript, Vite, TailwindCSS, and Framer Motion.

## Routes

- `/` Home
- `/about` About
- `/projects` Projects
- `/contact` Contact

## Local Development

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Content Editing

Update content in:

- `src/content/profile.json`
- `src/content/projects.json`
- `src/content/contact.json`

## Deployment

This project is configured for static deployment on Vercel.
